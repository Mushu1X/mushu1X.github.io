Tether-Net for Space Debris Capture
Pose Estimation and Vision-based Learning

High-fidelity Unreal Engine simulation for space debris capture using maneuverable Tether-Net system, including five camera sensors deployed on maneuverable units and chaser satellite.Utilized RCNN for precise debris detection and 6D pose estimation with camera sensors in Unreal Engine simulation. Implemented vision-based learning task to control movement of thrusters and fuel consumption for successful debris capture.

Quadruped - Object Detection,
Following and Avoidance

For small quadruped robots like those from Unitree, object detection is a primary task. In this project, we utilized the robotâ€™s built-in camera along with an external Intel RealSense Depth camera for object detection. The object detection model was trained using YOLO (You Only Look Once). Object following was implemented through waypoint navigation, which actively adjusts the robot's path as it detects objects and measures their distance. To prevent collisions with obstacles, the Bug2 algorithm was rigorously tested and integrated, enhancing the robot's ability to navigate complex environments safely
